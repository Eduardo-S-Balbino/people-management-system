# People Management System

Projeto desenvolvido em Python para gerenciamento de pessoas via terminal (CLI).

## Descrição

O People Management System é um sistema de cadastro de pessoas feito em Python, executado no terminal, que permite realizar operações de gerenciamento de dados com persistência em arquivo JSON.

O sistema foi desenvolvido com foco em praticar lógica de programação, funções, listas, dicionários, manipulação de arquivos e organização de projeto.

## Funcionalidades

- Cadastrar pessoa
- Listar pessoas cadastradas
- Buscar pessoa pelo nome
- Remover pessoa pelo nome
- Editar dados de uma pessoa cadastrada
- Impedir cadastro com nome vazio
- Impedir cadastro com nome duplicado
- Validar idade com entrada numérica
- Ignorar diferença entre letras maiúsculas e minúsculas na busca e remoção
- Salvar os dados em arquivo JSON
- Carregar os dados automaticamente ao iniciar o programa
