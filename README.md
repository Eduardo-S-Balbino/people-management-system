\# Sistema de Cadastro de Pessoas



Projeto desenvolvido em Python para gerenciamento de pessoas via terminal (CLI).



\---



\## Descrição



Este projeto consiste em um sistema de cadastro de pessoas que permite realizar operações básicas de gerenciamento de dados, como adicionar, listar, buscar e remover registros.



Os dados são armazenados em um arquivo JSON, garantindo persistência mesmo após o encerramento do programa.



\---



\## Funcionalidades



\- Cadastro de pessoas (nome e idade)

\- Listagem de todas as pessoas cadastradas

\- Busca de pessoa pelo nome

\- Remoção de pessoa pelo nome

\- Validação de entrada (nome não vazio e idade numérica)

\- Salvamento automático dos dados em arquivo JSON

\- Carregamento automático dos dados ao iniciar o programa



\---



\## Tecnologias utilizadas

\- JSON
